Selenium Version : 4.8.2


To run the project, follow these steps:

Navigate to the ML-Code folder.
Create two model files named trained_model_30min.pkl and vectorizer.pkl by running the Jupyter notebook.
Copy the generated files to the root folder where scrapper.py is located.
Run the scrapper.py file using the command python scrapper.py.
The program will execute and scrape Instagram for posts from the given account, and return all the comment sections.
A JSON file will be created, containing the data of bullying comments and not bullying comments.
Keep an eye on the console during the code execution, as after completion, a server will be created and a website will be hosted.
Click on the link in the console to open a website where you can see all the data related to bullying and non-bullying comments.
